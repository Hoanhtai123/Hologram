Step 1: Open FrontEnd 
Step 2: Open the hostlocal:3000
Step 3: Enter the command: node server.js
Step 4: Open BackEnd folder
Step 5: command: python detect_hand_handler.py

Guide to control the hologram object:
- Raise the Index finger to Display the hologram object
- Resize the hologram object by index, thumb finger
- Raise a hand to switch to state SUSPEND before you want to switch to another mode
- Raise two fingers to switch to SHIFTMODE: move the index finger to left or right
